# OpenManus Skills

## Web Research
Given a topic, search the web and compile findings.
```bash
/tmp/openmanus_fleet.sh "Research [TOPIC] and summarize the top 5 findings"
```

## Page Scraping
Navigate to a URL and extract structured data.
```bash
/tmp/openmanus_fleet.sh "Go to [URL] and extract all [DATA TYPE]"
```

## Documentation Gathering
Find and compile documentation for a technology.
```bash
/tmp/openmanus_fleet.sh "Find documentation for [TECH] and list the key APIs"
```

## Competitor Analysis
Research competing projects or tools.
```bash
/tmp/openmanus_fleet.sh "Compare [TOOL_A] vs [TOOL_B] based on web search results"
```

## Web Testing
Navigate a web app and report findings.
```bash
/tmp/openmanus_fleet.sh "Go to [URL] and test the [FEATURE] functionality"
```
